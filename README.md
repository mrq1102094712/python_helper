# python_helper
updated 2021.1.4   
crop and merge pdf use PyPDF2   
crop_pdf:   
    put .pdf file into ./ and run cropPDF.py    
    the .pdf file will be resize to A4(595.32, 841.92)    
    
merge_pdf:  
    put all .pdf files into ./ and run mergePDF.py  
    in ./ will be a new .pdf file which is merged.
